K-Highlights

K-Highlights is a viewer for Kindle highlights and clippings. The application is deployed as a website using HEROKU and published on PWA store.

Tecnologies Used:

  • REACT
  • Workbox for creating Service Worker and handling caching resources.
  • Bootstrap
